# Fitness Center Form Validation Website\

<img src="website-preview.PNG">

## Overview
This project is a multi-page fitness center website built with HTML and JavaScript. It includes a homepage with membership information and a linked application form that uses client-side validation to check user input before submission.

## Objective
The goal of this project was to create a basic website that combines page navigation, structured content, and form validation. It demonstrates how HTML can be used for layout and presentation, while JavaScript can be used to improve usability by validating form fields.

## Features
- Multi-page website structure
- Homepage with membership information and pricing
- Application form for new members
- Client-side JavaScript validation
- Validation for required fields, numeric fields, email format, and card details
- Internal navigation between pages

## Pages Included
- `index.html` → Homepage with fitness center information, membership tiers, and navigation
- `formValidation.html` → Membership application form with validation logic

## Validation Checks
The form validates:
- First and last name
- Address, city, and state
- ZIP code length and numeric format
- Phone number length and numeric format
- Email address format
- Membership package selection
- Credit card name and number
- Expiration date
- CVV
- Billing ZIP code
